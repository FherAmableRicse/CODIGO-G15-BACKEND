comandos de mongodb
1. mongo : inicia el mongodbshell
collection = tablas
2. show dbs : muestra las bases de datos
3. use cursos : define la base de datos cursos y para crear se tiene que insertar el primer registro
4. db.alumnos.insertOne({nombre:'fher amable',email:'famable@gmail.com'}) : inserta un registro  en la coleccion alumnos
5. show collections: muestra las colleciones de alumnos
6. db.alumnos.find() :  muestra todos los documentos(registros) de la collecion registro
7. db.alumnos.find().pretty() :  muestra todos los documentos(registros) de la collecion registro formateado
8. db.alumnos.insertMany() :  registra varios documentos
